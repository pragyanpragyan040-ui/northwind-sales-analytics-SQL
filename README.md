# Northwind Sales Analytics Project

## About the Project
This project was created to practice and apply SQL skills using the Northwind dataset. The goal was to analyze sales data and answer real business questions related to customers, products, categories, and employee performance.
By working with multiple connected tables, I gained hands-on experience with SQL joins, aggregations, and data analysis techniques commonly used by data analysts.
## Dataset Used
The project uses the Northwind database, which includes:
* Customers
* Orders
* Order Details
* Products
* Categories
* Employees
* Shippers
## Skills Applied
* SQL Joins
* Aggregate Functions (SUM, COUNT)
* GROUP BY
* ORDER BY
* CASE WHEN
* Revenue Analysis
* Business Problem Solving
## Analysis Performed
### Top Customers by Revenue
Identified the customers who generated the highest sales revenue.
### Best-Selling Products
Analyzed which products were sold the most based on quantity ordered.
### Category Performance
Compared product categories to determine which generated the highest revenue.
### Employee Performance
Evaluated employee contribution based on sales revenue generated.
### Shipping Analysis
Analyzed the revenue associated with different shipping companies.

* Working with relational databases

* Performing business-focused data analysis
* Extracting insights from raw transactional data
* Writing efficient SQL queries
